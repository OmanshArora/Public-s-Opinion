# Public-s-Opinion
PO is a social app based on web3 which allows user to give their opinions on the daily basis.
It declares public's opinion on certain topics and local problems which everyone is facing nearby,
and all the opinions are encrypted so that only fair result comes out.
In this app as there is no leader or organization to restrict or ban the users regarding the public's opinion,
it promotes democracy in society as everyone is free to give their opinion.
